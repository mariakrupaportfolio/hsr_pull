# HSR Pull Advisor

Honkai Star Rail Character Pull Recommendation Tool

## ✨ Setup for GitHub Pages Deployment

This project is configured to deploy automatically to GitHub Pages from the `/docs` folder!

### 🚀 Current Configuration:
- **Build Tool**: Vite + React
- **Build Output**: `/docs` folder  
- **Deployment**: GitHub Pages
- **Live URL**: https://mariakrupaportfolio.github.io/hsr_pull/

### 📋 To Enable GitHub Pages:

1. Go to your repository **Settings** → **Pages**
2. Configure:
   - **Source**: Deploy from a branch
   - **Branch**: `main` + `/docs` folder
3. Click **Save**
4. Done! GitHub will automatically deploy your site

Your site will be live at: `https://mariakrupaportfolio.github.io/hsr_pull/`

### 💻 Local Development

```bash
npm install      # Install dependencies
npm run dev      # Start dev server (localhost:5173)
npm run build    # Build for production (outputs to /docs)
npm run preview  # Preview the production build
```

### 🔄 Workflow

1. Edit files in `src/`
2. Run `npm run build` to create deployment files
3. Push to GitHub → Automatic deployment! ✅
